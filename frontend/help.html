How to Use the System

This page is for new staff members. Follow these steps to operate the reservation system safely.

1) Login

Open Login page, enter your username and password, and sign in.

2) Add Reservation

Go to Add page, fill the guest details, select room type and dates, then create reservation. Reservation number must be unique.

3) View Reservation

Use View page and enter reservation number to check saved details.

4) Billing

Use Billing page and enter reservation number. System calculates nights and total cost using room rates. You can print the bill.

5) Logout

Always logout after use to protect guest data.

Note: If pages redirect to login, your session expired or you are not logged in.
